離線pip安裝


1. 把整個pypkg資料夾放在:~/deid/sourceCode/webService/APP__/

2. 進入container:flask celery

3. cd app/devp/pypkg

4. pip install --no-index --find-links flasgger_files flasgger

5. pip install --no-index --find-links flask_cors_files flask_cors

6. pip install --no-index --find-links requests_files requests
